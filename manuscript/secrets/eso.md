## Setup

```bash
export GITOPS_APP=$(yq ".gitOps.app" settings.yaml)

cp $GITOPS_APP/external-secrets.yaml infra/.

git add . 

git commit -m "External Secrets"

git push

kubectl --namespace external-secrets get pods

# Wait until the Pods appear
```
